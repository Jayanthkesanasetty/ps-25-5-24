1 . write a program to print reverse of a number by using while loop
       -------code--------
        var a=15822
        var b=""
        var i=String(a).length-1;
        while(i>=0){
            b=b+String(a)[i]
            i--;
        }
        console.log(b);
    
        ------output------
        Reverse of 15822 is 22851
